# Multimodal Fake News Detection

## 📌 Overview
This project focuses on detecting fake news using both textual and visual information. It explores multimodal deep learning models and introduces explicit image–text consistency analysis using CLIP.

## 🚀 Key Features
- BiLSTM + Attention (Text)
- DistilBERT (Text)
- CLIP (Image)
- Multimodal Fusion
- Image–Text Consistency Score

## 🧠 Methodology
We compare:
- Classical RNN (BiLSTM)
- Transformer (DistilBERT)
- Multimodal + Consistency

## 📊 Experiments
- Text-only vs Image-only vs Multimodal
- Consistency score evaluation
- Ablation studies
